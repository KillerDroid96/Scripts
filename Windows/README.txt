This is for Windows 10 Hybrid OS
It depends on the Cygwin64 Toolchain to emulate Linux through a terminal. 
Msys64 can be used as an alternative to Cygwin64.

----------------------------------------------------------
Folder		    Location to put Folder
----------------------------------------------------------
Scripts   (C:\Users\mkell\Desktop\Coding\Github\Scripts)

----------------------------------------------------------
 File           Location to put File
----------------------------------------------------------
.bashrc   (C:\Toolchains\cygwin64\home\mkell)
motd.sh   (C:\Toolchains\cygwin64\etc\profile.d)
banner    (C:\Toolchains\cygwin64\bin)

The modified files make the functionality of Scripts folder the same as in Linux


Cygwin is structured like Linux under C:\Toolchains\cygwin64
-bin
-cygdrive
-dev
-etc
-home 
-lib
-sbin
-tmp
-usr
-var
